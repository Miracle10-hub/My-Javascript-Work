# JavaScript Lotto Simulator By Miracle
This repository contains 5 Node.js exercises for learning arrays, loops, randomness, input handling, and basic data structures.

Exercises included:
- Exercise1_LottoNumberPicker
- Exercise2_LottoStatisticsDashboard
- Exercise3_LuckyWordLotto
- Exercise4_LottoProfitCalculator
- Exercise5_LottoHistoryTracker

Run each by navigating to the folder and running `node solution.js`.
