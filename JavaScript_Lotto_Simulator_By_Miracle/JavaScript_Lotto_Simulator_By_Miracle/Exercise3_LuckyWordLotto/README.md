## Goal
Create a word-based Lotto game: user enters a 5-letter word and the system generates 5 random letters (A-Z) to compare.

## Instructions (summary)
1. Ask the user for a 5-letter word.
2. Convert it to uppercase and extract letters.
3. Generate 5 random letters A-Z (may repeat).
4. Compare how many letters match and display count and which ones.
5. Use a while loop to allow multiple rounds.
6. Generate a random score (1-100) per round and keep track of highest score and rounds played.

## How to run
```bash
node solution.js
```
