# Exercise 4 — Lotto Profit Calculator

## Goal
Calculate income and profit/loss for a lotto draw based on tickets sold, ticket price and total payout.

## Instructions (summary)
1. Ask for number of tickets sold and cost per ticket.
2. Ask for total payout (sum of all winnings).
3. Use arithmetic operations to calculate total income and profit (income − payout).
4. Display formatted messages using Math.round() and toFixed().
5. Allow repeating with a do–while loop until the operator exits.

## How to run
```bash
node solution.js
```
